What does this tool do?

This tool models particulate matter (PM2.5/PM10) dispersion from an urban concrete batching plant and compares two engineering control alternatives: enclosed aggregate storage with a fabric filter baghouse (Alt 1) and road/surface stabilization with wet suppression (Alt 2). It helps a user evaluate which intervention better reduces ambient PM concentrations reaching a neighboring residential receptor, and supports a structured multi-criteria decision.

Required inputs

Emission rate (g/s), wind speed (m/s), stack or release height (m), receptor height (m), lateral offset from centerline (m), Pasquill–Gifford atmospheric stability class (A–F), and control efficiency (%) for each alternative. All parameters are adjustable via the sidebar sliders.

Modeling approach

The tool implements a simplified Gaussian plume dispersion model following the Pasquill–Gifford framework (Seinfeld & Pandis, 2006). This approach is appropriate for a continuous, near-ground point source under steady-state meteorological conditions — representative of routine batching plant operations. Horizontal (σy) and vertical (σz) dispersion coefficients are computed from stability-class parameterizations and update in real time as inputs change.

Outputs and comparison structure

The tool produces: (1) centerline concentration profiles (µg/m³) vs. downwind distance (50–2000 m) plotted against the EPA 24-hr NAAQS for PM2.5 (35 µg/m³); (2) peak concentration summary cards for baseline and both alternatives; (3) a multi-criteria decision analysis (MCDA) scoring matrix across four dimensions; and (4) a rough order-of-magnitude cost comparison with implementation practicality notes.

How to interpret outputs

Lower concentration curves indicate greater PM reduction effectiveness. Exceedances of the 35 µg/m³ NAAQS line at receptor distances signal meaningful health risk. The MCDA weighted total reflects overall alternative performance — higher scores favor better alternatives under the stated criteria weights. Cost estimates are order-of-magnitude only; a higher total cost does not preclude selection if technical or health performance is substantially superior.

AI collaboration statement This tool was developed with substantial AI collaboration (Claude Sonnet, Anthropic, 2025) as part of CIVE 442 Tool 2. The Gaussian dispersion model, MCDA framework, cost estimates, interface design, and written content were co-generated and iteratively refined through AI-assisted development. Human judgment was applied throughout to validate modeling assumptions, select the scenario, interpret outputs, and make final design recommendations.
